# Minimal Todo API (.NET / ASP.NET Core)

Einfache REST API auf Basis von **ASP.NET Core Minimal APIs**.

Endpoints:

- `GET /` – Healthcheck
- `GET /todos` – Liste aller Todos
- `GET /todos/{id}` – Einzelnes Todo
- `POST /todos` – Neues Todo anlegen
- `PUT /todos/{id}` – Todo aktualisieren
- `DELETE /todos/{id}` – Todo löschen

Tech:

- .NET 8
- ASP.NET Core Minimal APIs

Start:

```bash
dotnet run
```
